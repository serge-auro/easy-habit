# Документация к модулю управления привычками

Модуль предназначен для работы с базой данных `easy_habit.db`, в которой хранится информация о пользователях, их привычках и статусах выполнения этих привычек.

## Функции

### `init_user(user_id)`

Инициализация нового пользователя в системе.

#### Аргументы:
- `user_id` (int): Уникальный идентификатор пользователя.

#### Возвращает:
- Ничего не возвращает. Функция записывает в базу данных нового пользователя с текущей датой регистрации.

### `init_habit(name, description)`

Добавление новой привычки в систему.

#### Аргументы:
- `name` (str): Название привычки.
- `description` (str): Описание привычки.

#### Возвращает:
- Ничего не возвращает. Функция записывает в базу данных новую привычку с заданным названием и описанием.

### `assign_habit(user_id, habit_id, frequency_name, frequency_count)`

Присвоение определённой привычки пользователю.

#### Аргументы:
- `user_id` (int): Идентификатор пользователя.
- `habit_id` (int): Идентификатор привычки.
- `frequency_name` (str): Название периода выполнения (например, "ежедневно").
- `frequency_count` (int): Количество повторений привычки за указанный период.

#### Возвращает:
- `message_text` (str): Сообщение о добавлении привычки пользователю.

### `list_habits(user_id)`

Вывод списка всех доступных привычек.

#### Аргументы:
- `user_id` (int): Идентификатор пользователя (пока не используется в функции).

#### Возвращает:
- `message_text` (str): Форматированное сообщение со списком всех привычек.

### `habit_status(user_id)`

Получение статуса всех активных привычек пользователя.

#### Аргументы:
- `user_id` (int): Идентификатор пользователя.

#### Возвращает:
- `output_message` (str): Сообщение со списком всех активных привычек пользователя или сообщение о том, что активных привычек нет.

### `edit_habit(user_id, habit_id, frequency_name, frequency_count)`

Изменение параметров привычки пользователя.

#### Аргументы:
- `user_id` (int): Идентификатор пользователя.
- `habit_id` (int): Идентификатор привычки.
- `frequency_name` (str): Новый период выполнения привычки.
- `frequency_count` (int): Новое количество повторений за период.

#### Возвращает:
- `message_text` (str): Сообщение об изменении параметров привычки.

### `delete_habit(user_id, habit_id)`

Удаление привычки у пользователя.

#### Аргументы:
- `user_id` (int): Идентификатор пользователя.
- `habit_id` (int): Идентификатор привычки.

#### Возвращает:
- `output_message` (str): Сообщение об удалении привычки.

### `mark_habit(user_id, habit_id, mark_date, count=1)`

Отметка выполнения привычки пользователем.

#### Аргументы:
- `user_id` (int): Идентификатор пользователя.
- `habit_id` (int): Идентификатор привычки.
- `mark_date` (str): Дата выполнения привычки в формате YYYY-MM-DD.
- `count` (int, optional): Количество раз выполнения привычки за один раз (по умолчанию 1).

#### Возвращает:
- `output_message` (str): Сообщение о количестве выполнений привычки и достижении цели по привычке.

## Общие замечания

Функции в этом модуле взаимодействуют с базой данных SQLite и предназначены для управления привычками пользователя. Они могут быть использованы для интеграции с Telegram ботом или другим интерфейсом пользователя.
